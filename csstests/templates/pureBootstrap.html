Bootstrap Page

This page is laid out with standard Bootstrap. Font, spacing and look are dependent on the bootstrap library. Elements are styled by adding class attributes to them. For example the nav-bar at the top of the page is a div with class navbar:

  <div class="navbar">
    <div class="navbar-inner">
      ...

When bootstrap css is included in the page, these standard classes are styled according to style rules defined there. For instance the navbar class has rules applied to it like this (from the included Bootstrap):

.navbar {
  *position: relative;
  *z-index: 2;
  margin-bottom: 20px;
  overflow: visible;
}

The table on this page is statically rendered, but we can imagine a customer wanting to include a Splunk result table instead. In order to render the result table, we need to add Splunk styles to the page.
