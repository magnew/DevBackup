Spunk Styles Added

On this page we have included markup for a Bubbles-style table and the Splunk styles required to render it. Unfortunatly, since Spunk styles are based on Bootstrap and use the same class names, the styles the user was relying on are overwritten and non-Splunk-related parts of the page get re-styled as a side effect. For example, the navbar class style from the Splunk bootstrap is different from the navbar class style from standard bootstrap:

.navbar {
  overflow: visible;
  margin-bottom: 16px;
  *position: relative;
  *z-index: 2;
}

These style overrides are not limited to controls like the navbar. As you can see many fairly general styles are affected including those governing spacing and fonts.

Ideally, to address this problem Splunk styles would not have the same names as the standard Bootstrap names. For example navbar would be renamed to spl-navbar. This way our styles would effectively be namespaced and thus only applied to controls where we specifically intend them to be.
